
⚠️ !!BEFORE FIRST USE!! ⚠️ 
 
Before launching TruStretch.exe normally, create at least one Resolution Profile in the Settings GUI.

  - Settings GUI -
  Hold SHIFT while launching TruStretch.exe to open the Settings GUI.
  In the Settings GUI you can add Resolution Profiles, select the profile you want to use, and configure your Default Resolution.
  For easier access to the Shift + Launch function, pin TruStretch.exe to Start or your Taskbar.
  You only need to hold SHIFT when opening the Settings GUI. Launch TruStretch.exe normally for everyday use.

  - Default Resolution - 
  Set your Default Resolution at the bottom of the Settings GUI. When your display is currently using a Profile Resolution, launching
  TruStretch.exe again will return it to your Default Resolution.

  - Important Notes - 
  **TruStretch DOES NOT CREATE CUSTOM RESOLUTIONS.**
  Resolution you add must already exist as a supported resolution in Windows, your GPU driver, or be a custom resolution you've already created.
  You can verify if a resolution is available by opening Settings > Display and checking the Display Resolution list.
  
 TruStretch is designed for fast resolution switching without opening Windows Display Settings.
 Launch TruStretch.exe normally to instantly switch between your Default Resolution and your selected Resolution Profile.
 No Windows menus, confirmation screens, or resolution revert prompts.

 Requirements
 ------------------
 • Windows 10/11
 • NirCmd must remain inside the Tools folder.
 • TruStretch.exe and the Tools folder must stay together.
